# Documentation Rules

You have access to local documentation in the `docs/` directory.
Before asking about or searching online for the following topics, check the local documentation first:

- **AI SDK**: `docs/ai_sdk.md` - Guide for building agents with Vercel AI SDK.
- **Diff-Match-Patch**: `docs/diff_match_patch.md` - Documentation for the `diff_match_patch` library.
- **ProseMirror**: `docs/prosemirror-docs/` - Comprehensive documentation for the ProseMirror text editor (API, Delta, Configuration, etc.).
- **Yjs**: `docs/yjs-docs/` - Documentation for the Yjs CRDT library (API, Ecosystem, etc.).

When working on features involving these technologies, refer to these local files.
